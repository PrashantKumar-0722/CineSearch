# CineSearch
"Hackathon Movie Hub: A responsive web application for finding popular and top-rated movies."

A dynamic, responsive web application developed during the [Hackathon Name] hackathon that allows users to discover, search for, and manage a personal watchlist of movies. This project was built to explore asynchronous API calls and front-end development best practices within a limited timeframe.

Technologies Used
Front-end: HTML, CSS, JavaScript, [e.g., React]
Back-end :- NodeJs
Database:- Mysql

Front-end:
HTML (HyperText Markup Language): Provides the foundational structure and content of web pages.
CSS (Cascading Style Sheets): Controls the visual presentation and styling of the HTML elements, including layout, colors, and fonts.
JavaScript: Adds interactivity and dynamic behavior to the web application, handling client-side logic and user interactions.
[e.g., React]: This indicates the use of a JavaScript library or framework for building user interfaces, with React being a prominent example. Other popular choices include Angular or Vue.js.
Back-end:
Node.js: A JavaScript runtime environment that allows server-side development using JavaScript, enabling the creation of scalable and efficient back-end services.
Database:
MySQL: A widely used open-source relational database management system (RDBMS) for storing and managing application data.
